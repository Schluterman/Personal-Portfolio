Katana Texture Mapping

This project illustrates my process for creating a texture map for my katana in 3ds max. General review for those that arn't familier with texture mapping: after creating a high or low poly object you must create a UV map which you can adjust for clearer mapping for your texture. This process would lead you to using photoshop to overlay whatever you want to appear on certain sections of your object when the texture is attached to the object.
